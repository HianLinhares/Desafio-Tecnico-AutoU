# Sistema de Classificação e Resposta Automática de Emails

Solução completa para automatizar a leitura e classificação de emails usando inteligência artificial.

## Funcionalidades

- **Classificação Automática**: Identifica se um email é Produtivo ou Improdutivo
- **Geração de Respostas**: Sugere respostas automáticas baseadas na classificação
- **Suporte a Múltiplos Formatos**: Aceita texto direto ou arquivos (.txt, .pdf)
- **Interface Moderna**: Design responsivo e intuitivo
- **Processamento com IA**: Utiliza técnicas de NLP para análise precisa

## Tecnologias Utilizadas

- **Backend**: Python com Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Processamento de Texto**: Regex, pré-processamento de texto
- **Hospedagem**: Compatível com Heroku, Vercel, AWS, etc.
